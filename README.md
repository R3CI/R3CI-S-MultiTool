## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/R3CI/R3CIS-Multitool.git
    ```

2. Navigate to the project directory:

    ```bash
    cd R3CIS-Multitool
    ```

3. Install dependencies:

    ```bash
    py main.py
    ```

4. Run the multitool:

    ```bash
    py main.py
    ```
